# halo集成wangEditor插件

## 介绍

wangEditor是一个开源的富文本编辑器,官网:https://www.wangeditor.com/

这个插件集成了wangEditor编辑器

## 部署方式

首先执行以下命令创建插件Jar包

```shell
./gradlew build
```

生成插件以后进入到build/libs文件夹,可以查看到生成的插件Jar包

进入Halo控制台界面,上传插件即可
